# Fix Zoxide Initialization Order

## Context

Zoxide is initialized in `~/.zshrc.local` (line 37-39), but `~/.zshrc` continues loading more things **after** sourcing `.zshrc.local`. The current load order in `~/.zshrc` is:

1. Kiro CLI pre block
2. ZSH-TOOL managed section (OMZ, plugins, aliases, exports, paths)
3. `source ~/.zshrc.local` ← **zoxide is initialized here**
4. `source zsh-tool.zsh`
5. PAI exports
6. Kiro CLI post block ← may override ZLE widgets
7. PAI alias
8. atuin env + `eval "$(atuin init zsh)"` ← likely overrides zoxide's hooks

Atuin's `init zsh` (line 72 of `~/.zshrc`) runs **after** zoxide and re-registers ZLE widgets, which trips zoxide's configuration validator. Zoxide requires being initialized last — after all other tools that modify ZLE/shell hooks.

## Root Cause

Zoxide is initialized mid-config in `.zshrc.local`, but both atuin and Kiro post-blocks run after it in `~/.zshrc`, overriding ZLE bindings zoxide set up.

## Fix

**Two changes:**

### 1. Remove zoxide init from `~/.zshrc.local`
Delete lines 35-39 from `~/.zshrc.local`:
```zsh
# ===== Directory Navigation =====
# zoxide - smart cd
if command -v zoxide >/dev/null 2>&1; then
  eval "$(zoxide init zsh --cmd cd)"
fi
```

### 2. Add zoxide init as the very last line of `~/.zshrc`
Append after line 72 (after `eval "$(atuin init zsh)"`):
```zsh
# zoxide - must be last (overrides cd; needs to run after all ZLE init)
if command -v zoxide >/dev/null 2>&1; then
  eval "$(zoxide init zsh --cmd cd)"
fi
```

## Files Modified

| File | Change |
|------|--------|
| `~/.zshrc.local` | Remove lines 35-39 (zoxide init block) |
| `~/.zshrc` | Append zoxide init after line 72 (after atuin init) |

Note: `~/.zshrc` is not in this repo (it's generated by `zsh-tool`), but the manually-added sections at the bottom (lines 54-73) are not overwritten by `zsh-tool-update` — so this addition is safe.

## Verification

```zsh
# Open a new terminal and check there's no zoxide warning
exec zsh

# Verify zoxide works
cd /tmp && cd - && z ~

# Confirm load order
echo "zoxide loaded: $(command -v __zoxide_z && echo yes || echo no)"
```
