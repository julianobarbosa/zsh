# zsh-tool Project Overview

> Generated by BMAD Document Project Workflow | 2025-12-13

## Executive Summary

**zsh-tool** is a comprehensive command-line tool for managing zsh shell configurations on macOS. It automates installation, configuration, plugin management, and maintenance of professional zsh environments with team-standard configurations.

## Quick Reference

| **Target Platform** | macOS 12+ |
| **Architecture** | Modular source-based library |
| **Status** | Production-ready (Epic 1-3 Complete) |

## Key Features

### Core Capabilities
- **Automated Installation**: Prerequisites detection, Oh My Zsh setup, plugin installation
- **Configuration Management**: Team-standard aliases, exports, PATH modifications
- **Backup & Restore**: Automatic backups with local/remote storage options
- **Self-Updating**: Keep the tool and all plugins up to date
- **Git Integration**: Version control for dotfiles

### Advanced Integrations
- **Atuin**: Enhanced shell history with searchable SQLite database
- **Amazon Q Developer CLI**: AI-powered command completions
- **Compatibility Layer**: Automatic configuration for Atuin + Amazon Q coexistence

## Architecture Principles

- **Idempotent**: Safe to run operations multiple times
- **Reversible**: Automatic backups before all changes
- **Modular**: Clear separation of concerns
- **Well-logged**: Progress indicators and helpful error messages
- **Non-intrusive**: Preserves user customizations

## Getting Started

```bash
# Clone and install
git clone https://github.com/julianobarbosa/zsh.git
cd zsh
zsh install.sh

# Reload shell and run setup
exec zsh
zsh-tool-install
```
